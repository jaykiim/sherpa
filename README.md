## What is Sherpa

Sherpa 는 조직의 성과와 목표 관리를 위한 프레임 워크 중 하나인 [OKR](https://www.salesforce.com/kr/hub/organization/what-is-okr/) 을 개인의 목표 달성에 적용할 수 있도록 돕는 툴입니다.
